My Small Projects

Blood Donation Record System

It is a web-based application designed to streamline the management of blood donation records using front-end technologies and a robust DBMS (Database Management System). This system enables users to efficiently record, update, and retrieve donor information, blood type details, and donation history. It ensures a user-friendly interface, while the DBMS tables provide a secure and organized structure for data storage, ensuring accuracy and accessibility. This project aims to enhance the efficiency and reliability of blood donation management for hospitals and blood banks.

Career Directions

Career Directions is a desktop application developed using Java and the NetBeans IDE, designed to assist users in exploring and planning their career paths. The application offers personalized career guidance by assessing users' skills, interests, and goals, and providing tailored career recommendations. With an intuitive interface, Career Directions simplifies the process of career planning, helping users make informed decisions about their professional future.

Flappy Bird Game

Flappy Bird Game is a Python-based project that replicates the popular arcade-style game where players navigate a bird through a series of pipes without hitting them. Developed using Python's Pygame library, this project features simple yet engaging graphics, smooth gameplay mechanics, and an intuitive control system. The game challenges players' timing and coordination, providing an entertaining and nostalgic gaming experience.

Student Management System

The Student Management System is a simple project developed using basic C language, designed to manage student records efficiently. It allows users to add, view, modify, and delete student information such as names, roll numbers, and grades. This console-based application provides a straightforward and effective solution for managing student data, making it an ideal project for learning fundamental programming concepts in C.
